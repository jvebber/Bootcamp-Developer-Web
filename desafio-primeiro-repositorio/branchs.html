Comandos para realizar um branch

  1. git branch: para verificar as branchs existentes
  2. git checkout -b nome_branch: para criar uma nova branch
  3. Após trabalhar normalmente no projeto (na branch), deve-se dar um commit normal
  4. Após isso, dar um push da branch no repositório remoto: git push -u origin nome_branch
  5. Depois, deve-se voltar para voltar para a branch master: git checkout master
  6. Aí é só dar um merge para aplicar na branch master as alterações da branch criada: git merge nome_branch
  7. Por fim, só atualizar a branch master: git push origin master
  8. git branch -d nome_branch: para apagar a branch do repositório local
  9. git push origin --delete nome_branch: para apagar a branch do repositório remoto
